# todolist-cli

[Source code](https://github.com/HimC29/todolist-cli)

## Installation

1. Add the repository to your system:

```
echo "deb [trusted=yes] https://HimC29.github.io/todolist-cli-deb stable main" | sudo tee /etc/apt/sources.list.d/todolist-cli.list
```

2. Update apt to see the new package:

```
sudo apt update
```

3. Install the package:

```
sudo apt install todolist-cli
```

4. All together:

```
echo "deb [trusted=yes] https://HimC29.github.io/todolist-cli-deb stable main" | sudo tee /etc/apt/sources.list.d/todolist-cli.list
sudo apt update
sudo apt install todolist-cli
```
